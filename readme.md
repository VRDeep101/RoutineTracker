# ⚡ GLOW UP OS

Personal transformation dashboard. Runs in your browser. No account. No internet needed.
Public Link: https://vrdeep101.github.io/RoutineTracker/

---

## How to Run

1. Extract the ZIP anywhere (Desktop is fine)
2. Open the `glowup` folder
3. Double-click `index.html` — opens in Chrome/Firefox — done

All data saves automatically in your browser.

## AI Session Setup (Free)

1. Go to [console.groq.com](https://console.groq.com) — sign up free
2. Click **API Keys** then **Create API Key** then copy it
3. Open app → **AI Session** page → paste key → Save
4. Done. Talk to your AI coach daily for 30 mins.

## Rules

1. **Never miss gym twice in a row** — one day is life, two breaks the habit
2. **2 hours hacking study is sacred** — phone silent, every day
3. **Track everything** — if it's not logged, it didn't happen
4. **Build in silence** — don't post about the grind, let results speak
5. **Sunday review every week** — rate each domain 1–10, plan next week
6. **Hard things first** — gym before phone, study before entertainment
